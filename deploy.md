
# 柒年树洞

#### 介绍
用 VuePress 搭建的博客，用来分享学习经验

#### 软件架构
文档：[VuePress](https://www.vuepress.cn/) Vue 驱动的静态网站生成器

主题：[vuepress-theme-reco](https://vuepress-theme-reco.recoluan.com/)

#### 安装教程

1.  安装 theme-cli 
```
yarn global add @vuepress-reco/theme-cli
```
2.  安装依赖
```
yarn install
```
3.  本地运行
```
yarn dev
```
4.  本地打包
```
yarn build
```
5.  自动部署
```
yarn deploy
```
