## 文件夹操作命令
创建 test 文件夹
``` sh
mkdir test
```
删除 test 文件夹
``` sh
rd test
```
## 文件操作命令
创建 test.txt 文件
``` sh
echo > test.txt
```
创建带内容的文件
``` sh
echo '# Hello test' > README.md
```
查看 README.md 文件内容
``` sh
type README.md
```
删除 test.txt 文件
``` sh
del README.md
```
